NAME

xbee_end  

SYNOPSIS

#include <xbee.h>

int xbee_end(void);  

DESCRIPTION

The xbee_end() function can be used to end a libxbee instance.

It will return the XBee unit to its initial API mode (if a variant of xbee_setupAPI() was used).

All connections, queued packets and internal structures will be removed from memory, and any open files or devices will be closed.
