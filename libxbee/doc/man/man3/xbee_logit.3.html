NAME

xbee_logit, xbee_logitf  

SYNOPSIS

#include <xbee.h>

void xbee_logit(char *str);

void xbee_logitf(char *format, ...);  

DESCRIPTION

xbee_logit() provides the very simple ability to add your own logging to the libxbee log (logging must be enabled by using a xbee_setuplog() variant).

xbee_logitf() provides more complex logging support. The parameters exactly match that of printf(2).
